# DX Tools Plugin

Developer experience optimization, debugging, and productivity enhancement.

## Installation

```bash
/plugin install dx-tools@cameronsjo
```

## Commands

### `/five`

Apply Five Whys root cause analysis to systematically investigate issues.

**Usage:**
```bash
/five "Why are builds failing?"
```

### `/optimize`

Analyze code performance and propose optimization improvements.

**Usage:**
```bash
/optimize
```

## Agents
