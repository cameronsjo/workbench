# Data Science Plugin

SQL optimization, ML pipelines, and data engineering.

## Installation

```bash
/plugin install data-science@cameron-tools
```

## Agents

### data-scientist

SQL, BigQuery, and data analysis expertise.

**Capabilities:**
- Complex SQL queries
- BigQuery optimization
- Data insights
- Statistical analysis

### data-engineer

ETL pipelines and data infrastructure.

**Capabilities:**
- ETL/ELT design
- Data warehouse architecture
- Streaming (Kafka)
- Spark jobs
- Airflow DAGs

### data-analyst

Quantitative analysis and metrics.

**Capabilities:**
- Trend analysis
- Performance metrics
- Benchmarking
- Statistical evaluation
- Reporting

### ml-engineer

ML model deployment and serving.

**Capabilities:**
- Model integration
- A/B testing
- Performance monitoring
- Feature engineering
- TensorFlow/PyTorch

### mlops-engineer

ML infrastructure and automation.

**Capabilities:**
- MLflow setup
- Kubeflow integration
- Experiment tracking
- Model registry
- Automated retraining

### quant-analyst

Financial modeling and trading algorithms.

**Capabilities:**
- Backtesting strategies
- Portfolio optimization
- Risk metrics
- Statistical arbitrage

### sql-expert

Complex SQL optimization and design.

**Capabilities:**
- Query optimization
- Index design
- Performance tuning
- Complex joins

## Skills

### ai-integration

LLM and AI API integration patterns.

**Covers:**
- API integration
- RAG systems
- Vector search
- Agent orchestration

## Example Usage

### Optimize SQL

```
"Use sql-expert to optimize this slow query"
```

### Build Data Pipeline

```
"Use data-engineer to design an ETL pipeline for our analytics"
```

### ML Deployment

```
"Use ml-engineer to implement model serving for our recommendation system"
```

### Financial Analysis

```
"Use quant-analyst to backtest this trading strategy"
```
