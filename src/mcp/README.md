# MCP Plugin

MCP (Model Context Protocol) server development toolkit.

## Installation

```bash
/plugin install mcp@cameronsjo
```

## Skills

### mcp-development

Complete guide for building MCP servers:

- Architecture patterns
- Client/server development
- Server discovery
- Sampling with tools
- PII sanitization patterns

### mcp-tools-as-code

Define MCP tools programmatically with TypeScript:

- Tool generation from code
- Transport handling
- Example workflows
