### Is this statement accurate?

- [ ] Yes, this is accurate
- [ ] No, this is aspirational (revise to be more realistic)

### Is this statement specific?

Could this apply to anyone in my role, or is it distinctive to me?

- [ ] Distinctive to me

### Is this statement evidence-based?

Can I point to multiple sources that support this statement?

- [ ] Yes, supported by 3+ sources
- [ ] No, needs more evidence (gather more input)
