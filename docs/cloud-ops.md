# Cloud Ops Plugin

Cloud infrastructure, Kubernetes, and DevOps automation.

## Installation

```bash
/plugin install cloud-ops@cameron-tools
```

## Agents

### cloud-architect

AWS/Azure/GCP infrastructure design and optimization.

**Capabilities:**
- Multi-cloud architecture
- Infrastructure as Code (Terraform)
- Cost optimization
- Auto-scaling patterns
- Multi-region deployment
- Serverless architecture

### terraform-specialist

Infrastructure as Code with Terraform.

**Capabilities:**
- Module design
- State management
- Multi-environment setup
- Provider configuration
- Best practices

### devops-troubleshooter

Production issue debugging and resolution.

**Capabilities:**
- Log analysis
- Monitoring integration
- Root cause analysis
- Incident response
- System recovery

### deployment-engineer

CI/CD pipelines and containerization.

**Capabilities:**
- GitHub Actions workflows
- Docker configuration
- Kubernetes deployment
- Pipeline optimization

### incident-responder

Production incident handling with urgency.

**Capabilities:**
- Rapid diagnosis
- Hotfix implementation
- Communication coordination
- Post-mortem documentation

## Example Usage

### Design Cloud Architecture

```
"Use cloud-architect to design a multi-region deployment for high availability"
```

### Infrastructure as Code

```
"Use terraform-specialist to create modules for our AWS infrastructure"
```

### Debug Production Issues

```
"Use devops-troubleshooter to investigate the current outage"
```

### Setup CI/CD

```
"Use deployment-engineer to create GitHub Actions workflows"
```

### Handle Incidents

```
"Use incident-responder to coordinate response to the production issue"
```
