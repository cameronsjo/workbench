# TypeScript Toolkit Plugin

TypeScript, React, and Next.js development patterns with modern best practices.

## Installation

```bash
/plugin install typescript-toolkit@cameronsjo
```

## Agents

### typescript-expert

Write type-safe TypeScript with advanced type system features.

**Capabilities:**
- Advanced generics and utility types
- Discriminated unions
- Conditional types
- Template literal types
- Type inference patterns
- Migration from JavaScript

### frontend-developer

Build Next.js applications with React components, shadcn/ui, and Tailwind CSS.

**Capabilities:**
- Next.js App Router
- Server/Client Components
- React patterns
- shadcn/ui integration
- Tailwind CSS
- SSR/SSG optimization

### javascript-expert

Modern ES6+ JavaScript and Node.js development.

**Capabilities:**
- ES6+ features
- Async patterns
- Node.js APIs
- Performance optimization
- Testing with Jest/Vitest

### ui-ux-designer

User interface and experience design.

**Capabilities:**
- Design systems
- Accessibility standards
- User research patterns
- Wireframing
- Prototyping

### golang-expert (bonus)

Go development for backend services often paired with TypeScript frontends.

## Example Usage

### TypeScript Development
