# Roadmap

## Ideas

| Item | Priority | Effort | Details |
|------|----------|--------|---------|
| Asset usage analytics | medium | medium | Track which assets are most used across plugins |
| Auto-update notifications | low | medium | Notify when installed plugins have updates |
| Plugin templates | low | small | Starter templates for common plugin patterns |
| Plugin versioning system | high | large | Semantic versioning, changelog generation |
| Web-based plugin browser | medium | large | Browse and preview plugins without CLI |
