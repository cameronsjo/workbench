# Obsidian PKM Plugin

Personal knowledge management with Obsidian markdown and vault organization.

## Installation

```bash
/plugin install obsidian-pkm@cameron-tools
```

## Agents

### connection-agent

Link suggestions for knowledge graphs.

**Capabilities:**
- Entity-based connection analysis
- Keyword overlap detection
- Orphaned note identification
- Link suggestion generation

### moc-agent

Maps of Content generation and organization.

**Capabilities:**
- MOC creation
- Navigation hub design
- Content organization
- Linking structure maintenance

### tag-agent

Tag taxonomy management and normalization.

**Capabilities:**
- Tag normalization
- Hierarchy organization
- Duplicate consolidation
- Taxonomy design

## Skills

### obsidian-markdown

Obsidian-optimized markdown patterns for GitHub compatibility.

**Covers:**
- GFM-compatible syntax
- Frontmatter structure
- Linking patterns
- Mermaid diagrams
- Callout syntax
- Dataview queries

## Example Usage

### Create Connections

```
"Use connection-agent to find linking opportunities in my vault"
```

### Generate MOCs

```
"Use moc-agent to create a Map of Content for my programming notes"
```

### Organize Tags

```
"Use tag-agent to clean up and organize my tag taxonomy"
```

### Write Notes

```
"Following obsidian-markdown patterns, create a note about..."
```
