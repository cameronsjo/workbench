# Research Tools Plugin

Comprehensive research, academic sources, and information gathering.

## Installation

```bash
/plugin install research-tools@cameron-tools
```

## Agents

### comprehensive-researcher

Multi-source research with verification and structured reports.

**Capabilities:**
- Multi-source research
- Cross-verification
- Structured reports
- Citation management
- Balanced analysis

### academic-researcher

Scholarly sources and literature review.

**Capabilities:**
- Academic paper discovery
- Literature reviews
- Citation analysis
- Research trends
- Scientific evidence

### technical-researcher

Code and documentation analysis.

**Capabilities:**
- Repository analysis
- API documentation review
- Implementation research
- Technical comparison

### search-specialist

Advanced search techniques and optimization.

**Capabilities:**
- Query optimization
- Result filtering
- Multi-source verification
- Competitive analysis

## Skills

### remembering-conversations

Search previous Claude Code conversations for context and decisions.

**Capabilities:**
- Semantic search across past conversations
- Find historical decisions
- Recover context after `/clear`
- Pattern recognition across sessions

## Example Usage

### Comprehensive Research

```
"Use comprehensive-researcher to investigate best practices for rate limiting"
```

### Academic Research

```
"Use academic-researcher to find papers on distributed consensus algorithms"
```

### Technical Research

```
"Use technical-researcher to analyze how other projects implement caching"
```

### Find Past Decisions

```
"Use remembering-conversations to find why we chose PostgreSQL"
```
