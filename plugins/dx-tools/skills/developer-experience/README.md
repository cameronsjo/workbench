# Developer Experience Skill

Optimize development workflows, reduce friction, and make development joyful and productive.

## Quick Start

Claude automatically invokes this skill when you mention:
- Developer experience, DX, workflow optimization
- Project setup, environment configuration
- Build/test performance issues
- IDE configuration, git hooks
- Onboarding friction, manual tasks

## What's Included

Comprehensive guidance for:
- Sub-5-minute environment setup
- IDE configuration (VS Code, etc.)
- Git hooks with Husky
- Task automation (Makefile, scripts)
- Error messages and feedback
- Performance optimization
- Documentation improvements
- Onboarding automation

## Usage Examples

```
"Setup a new project with automated environment setup"
"Our onboarding takes 2 hours. Help me reduce it to under 15 minutes"
"Add git hooks to catch issues before commit"
"Create a Makefile for common development tasks"
```
