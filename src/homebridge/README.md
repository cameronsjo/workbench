# Homebridge Plugin

Expert-level Homebridge plugin development assistance - from scaffolding through debugging.

## Usage

```
/homebridge
```

## Capabilities

- **New plugin scaffolding** - JS or TS, proper package.json, config schema stub
- **Accessory implementation** - Correct HAP service/characteristic mappings
- **Config schema authoring** - Validation, UI hints, conditional fields
- **Log analysis** - Common error patterns, debugging workflows
- **Verified publisher** - NPM setup, verification process
- **Child bridges** - When and how to use isolation
- **HAP Inspector** - Traffic analysis and debugging

## Components

### Skill: `/homebridge`

Loads expert context and guides your work. Detects JS vs TS from your project.

### Agent: `homebridge-explorer`

Research agent for deeper dives - HAP lookups, log analysis, pattern discovery.
